# Wine Vybe (wine-vybe)
Wine Vybe is a wine, beer, and liquor database API that provides information on wine regions, grape varieties, tasting notes, food pairing, awards, and brewery and distillery information. The API enables developers to build applications that help users discover and select wine and beverage products.

## APIs

### Wine Vybe API
The Wine Vybe API provides access to a wine, beer, and liquor database including wine regions, grape varieties, tasting notes, food pairing recommendations, awards, producer information, brewery and distillery details, and custom-built wine tasting profiles.
